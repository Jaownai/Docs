DataServiceClient

Client-side API for reading replicated player data and reacting to changes. All values are a mirror of what the server last wrote — no DataStore access happens on the client.

💻  Client  ·  require(DataService).client
Data
get
DataServiceClient:get(path?: Path) → any

Returns the local player's in-memory data. Omit path for the full data table. path accepts a string key or an array of strings for nested access.

Note: Data is available immediately after the server's initial replication fires. Use fetch if you need to wait for it.
ParameterTypeDescription
path optionalstring | {string}Key or nested path. Omit for the full data table.
Returns any
Lua 
-- Full snapshot
local data = DataServiceClient:get()

-- Single field
local coins = DataServiceClient:get("Coins")

-- Nested path
local wins = DataServiceClient:get({"Stats", "Wins"})
fetch
yields
DataServiceClient:fetch(path?: Path) → any

Yields until the initial data replication from the server has arrived, then returns data at path (or the full table if omitted). Safe to call at the very top of a LocalScript.

ParameterTypeDescription
path optionalstring | {string}Key or nested path. Omit for the full table.
Returns any
Lua 
-- Safe at the top of a LocalScript before data arrives
local coins = DataServiceClient:fetch("Coins")
print(coins)
Reactive
onChange
DataServiceClient:onChange(path: Path, callback: (new: any, old: any) → ()) → RBXScriptConnection

Fires callback whenever the value at path changes due to a server replication. Receives the new value and the previous value. Returns a connection that can be :Disconnect()-ed.

Note: Subscribes to the exact path. Watching a parent path does not fire when a child key changes — subscribe to the leaf you care about.
ParameterTypeDescription
pathstring | {string}Key or nested path to watch.
callback(new: any, old: any) → ()Called with the new and previous values.
Returns RBXScriptConnection
Lua 
local conn = DataServiceClient:onChange("Coins", function(new, old)
    print("Coins:", old, "→", new)
end)

-- Disconnect when no longer needed
conn:Disconnect()
onChanged
alias
DataServiceClient:onChanged(path: Path, callback: (new: any, old: any) → ()) → RBXScriptConnection

Alias for onChange. Identical behaviour.

Lua — watch multiple fields 
DataServiceClient:onChanged("Coins", function(new, old)
    -- update coin display
end)

DataServiceClient:onChanged("Level", function(new, old)
    -- update level display
end)

DataServiceClient:onChanged("Inventory", function(new, old)
    -- refresh inventory UI
end)
